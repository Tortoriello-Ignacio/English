FAQs

Preguntas frecuentes

Guía rápida para entender cómo usar el entrenador de forma ordenada.

¿Dónde se guarda mi información?

En el almacenamiento local del navegador. Si cambiás de navegador o dispositivo, esos datos no viajan automáticamente.

¿Cómo se registra el tiempo de estudio?

Se suma cuando hay interacción reciente con la página. Si la dejás abierta sin usarla, el contador se pausa.

¿El calendario se vincula al progreso?

Sí. Los días activos quedan marcados y el tiempo diario se refleja en el calendario mensual.

¿Puedo usarlo sin internet?

La mayoría de las funciones son locales. Las voces dependen de las voces disponibles en tu navegador/sistema.

Términos y privacidad

Uso local y control de datos

Texto base para que puedas adaptar si luego publicás el proyecto.

Esta herramienta está pensada como entrenador personal de inglés. Los apuntes, progreso, calendario y prácticas se almacenan localmente en el navegador mediante localStorage.

No se envían datos a un servidor desde esta versión estática. Si más adelante agregás backend, login o IA real, conviene actualizar esta sección con una política formal.
